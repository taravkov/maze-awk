# *Maze*

Terminal game for *nix systems. ಠ_ಠ

![alt text][logo]

[logo]: https://raw.githubusercontent.com/taravkov/maze-awk/master/maze.png "Maze"

* new maze is generated each time you play
* supports different maze sizes, defaults to 8x8
* written in AWK

### How to play:
```sh
chmod u+x maze.awk
maze.awk
```

Keys: WASDQ
